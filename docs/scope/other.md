# Вспомогательные функции
Некоторые вспомогательные функции бота

## Спам фразой
`!spam number value`
- `number` - сколько раз отправится команда
- `value` - ваша фраза

## Проверка бота
`!ping`

## Помощь
`!help`

## Отправить репорт
`!report value`
- `value` - ваше сообщение
